---------------------------------------------------------------------- GIT COMMANDS ---------------------------------------------------------------------
git inint => make a git repository
git add . => add all files to stage 
git status => see status of files
git commit -m "messsage" => commit files of stage with a message
git rm --cached -r . => remove all file from stage
git diff => see what is changed in files
git log => see commits with messages
git log -p => see commits with changes
git log --oneline => see comits in one line brifly
git restore . => restore change to last commit
git restore --staged . => unstaged all of files in stage and can restore
git reset --hard 5fc0f1aaa9321f2c0886b44a0bb361d84b9b4c38 => rest to that id commit 
git branch branchName => make a new branch
git branch -a => show branch list
git branch -d branchName => for delete a branch
git checkout branchName => switch to another branch 
git merge branchName => first most going to destination target
git stash => make stash from new changes 
git stash save "message of this stash"
git stash list => show list of stash 
git stash clear => clear all stashs
git stash drop stashName => delete a stash 
git stash show -p => show change in stash 
git stash apply "stashName" => add a stash without delete the stash 
git stash pop "stashName" => add a stash with delete after add 
git remote add origin https://github.com/MehhdiMarzban/git.git => add address
git push -u origin main => upload a branch to github
git pull origin => get last change from github
git pull => this change from github
git clone url nameOfFolder => add a project from github
